About

온라인 쇼핑몰 고객 세분화 분석

2023년 10월 - 11월

🎯 문제 (Problem)

매출 구조 파악 부족으로 타겟팅된 마케팅 캠페인이 비효율적. 어떤 고객 세그먼트가 가장 수익성 높고, 어떤 세그먼트는 이탈 위험이 높은가?

🛠️ 접근 (Approach)

1단계: SQL로 지난 12개월 구매 데이터 추출 (100만 건 거래)
2단계: Python에서 RFM 분석 (Recency, Frequency, Monetary)
3단계: K-means 클러스터링으로 5개 세그먼트 도출
4단계: Tableau 대시보드로 세그먼트별 KPI 시각화

📊 결과 (Result)

  • VIP 고객(상위 5%): 전체 매출의 55% 차지
  • 위험 고객(구매 중단): 이탈률 88% → 개입 후 30% 개선
  • 신규 고객: 첫 구매 후 3개월 내 이탈률 65% (리텐션 전략 필요 식별)
  • 마케팅 ROI 개선: 세그먼트 기반 타겟팅으로 캠페인 효율 34% 증대

💻 기술 스택

SQL (MySQL) Python (pandas, scikit-learn) Tableau Excel (Pivot)

고객 이탈 예측 모델링

2023년 08월 - 09월

🎯 문제 (Problem)

고객 이탈을 사후에 파악하지 말고 조기에 예측하여 리텐션 캠페인을 선제적으로 실행할 수 있는가? 이탈 고위험군을 식별하는 모델 필요.

🛠️ 접근 (Approach)

데이터 준비: 12개월 고객 데이터 (30,000명), 이탈 라벨 생성
특성 엔지니어링: 구매 빈도, 최근 구매 경과일, 평균 구매액, 구매 주기, 카테고리 다양성 등 15개 피처
모델 검토: Logistic Regression, Random Forest, XGBoost 비교
최종 선정: Logistic Regression (AUC 0.81, 해석 가능성 우수)

📊 결과 (Result)

  • 모델 성능: AUC 0.81, 정밀도 0.76, 재현율 0.71
  • 상위 20% 고위험 고객: 실제 이탈률 62% (베이스라인 28% 대비 2.2배)
  • 리텐션 캠페인 타겟팅: 월 3,000명 대상, 복귀율 18% 달성
  • 예상 효과: 연간 고객생명가치(LTV) 15% 증대 ($450만 추정)

💻 기술 스택

Python (scikit-learn, pandas) SQL Jupyter Notebook matplotlib, seaborn

🛠️ 접근 (Approach)

  • 의사결정 시간: 수동 집계 2시간 → 실시간 조회로 즉시 가능

2023년 04월 - 05월

🎯 문제 (Problem)

마케팅팀이 어떤 컨텐츠가 실제 매출로 이어지는지, 어떤 채널이 가장 효율적인지 파악 부족. 컨텐츠 투자 의사결정이 데이터 기반이 아님.

🛠️ 접근 (Approach)

데이터 통합: Google Analytics, 소셜 미디어, 웹사이트 이벤트 데이터 병합
속성분석: 고객 여정 추적, 각 컨텐츠의 기여도 측정
코호트 분석: 컨텐츠 노출 시점별 고객 그룹 추적
통계검정: 채널/형식별 성과 차이의 통계적 유의성 검증

📊 결과 (Result)

  • 이메일 캠페인: 개방율 28%, 클릭율 4.2%, 전환율 1.8% (산업 평균 대비 25% 높음)

  • 블로그: 검색 최적화 후 월 방문자 8,000명 → 18,000명 증가
  • 마케팅 ROI: 채널별 최적 예산 배분으로 전체 ROI 28% 개선

Analysis Process

데이터 분석 프로세스를 단계별로 시각화합니다.

1

문제 정의 (Define)

비즈니스 질문을 명확히 하고, 성공 지표(KPI)를 결정합니다. 이 단계가 분석의 방향을 결정합니다.

  • 스테이크홀더와의 인터뷰
  • 비즈니스 목표 수립
  • KPI/성공 지표 정의
→
2

데이터 수집 (Collect)

필요한 데이터를 데이터베이스, API, 또는 기타 소스에서 추출합니다. 데이터 품질 확인이 중요합니다.

  • SQL 쿼리 작성
  • API 연결
  • 데이터 추출 및 검증
→
3

탐색 분석 (Explore)

데이터의 분포, 이상치, 결측치, 상관관계를 파악합니다. 패턴과 인사이트의 초기 발견 단계입니다.

  • 기술통계 분석
  • 데이터 분포 확인
  • 이상치/결측치 처리
→
4

분석 (Analyze)

통계 검정, 회귀분석, 클러스터링, 머신러닝 등 정교한 분석 기법을 적용합니다. 가설을 검증하고 인사이트를 도출합니다.

  • 가설 검정
  • 회귀/분류 모델
  • 세그멘테이션
→
5

시각화 (Visualize)

복잡한 분석 결과를 차트, 그래프, 대시보드로 변환하여 직관적으로 이해할 수 있도록 합니다.

  • 차트 설계
  • 대시보드 구축
  • 스토리텔링
→
6

제안 (Recommend)

분석 결과를 기반으로 실행 가능한 제안을 작성합니다. 예상 임팩트와 함께 우선순위를 제시합니다.

  • 액션 계획 수립
  • 임팩트 추정
  • 리포팅
