## 📌 Project 1. 온라인 쇼핑몰 구매 데이터 분석

> **고객 행동 분석을 통해 매출 증대 전략 도출**

### 🔍 Problem

* 어떤 고객이 매출에 가장 크게 기여하는가?
* 재구매율을 높일 수 있는 고객군은 누구인가?

### 🧩 Data

* 주문 데이터 약 100만 건
* 고객 / 주문 / 상품 테이블 기반 분석

### 🛠 Tools

`Python · SQL · Excel · Tableau`

### 📊 Process

1. **SQL**로 핵심 지표 추출 (구매 횟수, 총 매출, 객단가)
2. **Python EDA**로 분포 및 이상치 분석
3. **Excel Pivot**으로 카테고리별 매출 구조 분석
4. **Tableau 대시보드**로 KPI 시각화

### 📈 Key Insights

* 상위 20% 고객이 전체 매출의 **62%** 차지
* 재구매 고객의 평균 구매 금액이 **1.8배 높음**
* 금요일 18~21시 주문 집중 현상 확인

### 💡 Actionable Insight

* VIP 고객 전용 쿠폰 전략 제안
* 금요일 저녁 타임세일 캠페인 설계
* 고재구매 상품 중심 추천 로직 개선

## 📌 Project 2. 고객 이탈 예측 분석

> **이탈 가능 고객을 조기에 식별하여 유지 전략 수립**

### 🔍 Problem

* 어떤 고객이 이탈할 가능성이 높은가?

### 🛠 Tools

`Python (sklearn), SQL`

### 📊 Approach

* Feature Engineering (최근 구매일, 구매 빈도, 평균 금액)
* Logistic Regression 기반 이탈 예측 모델링

### ✅ Result

* AUC Score: **0.81**
* 이탈 가능 상위 20% 고객군 도출

### 💡 활용 방안

* 이탈 위험 고객 대상 할인/리텐션 캠페인
* CRM 타겟 마케팅 고도화
